# 🔐 Authentication Backend

RESTful backend API for the Authentication Dashboard built with Node.js, Express.js, MongoDB, and JWT.

---

## 🚀 Features

- User Registration API
- User Login API
- JWT Authentication
- Password Hashing
- MongoDB Integration
- Protected Routes
- RESTful API Design

---

## 🛠 Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- dotenv

---

## ⚙️ Installation

```bash
git clone https://github.com/Guruio/auth-backend.git

cd auth-backend

npm install

npm start
```

---

## 🔐 Environment Variables

Create a `.env` file.

```env
MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

---

## 📂 API Endpoints

### Authentication

POST /signup

POST /login

GET /protected
